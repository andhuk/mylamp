## Використання в коді

```javascript
// Приклад 1: Пункт меню
var icon = '<svg height="36" viewBox="0 0 24 24" width="36" xmlns="http://www.w3.org/2000/svg">' +
    '<path d="M12 4V1L8 5l4 4V6c3.31 0 6 2.69 6 6..." fill="currentColor"/>' +
    '</svg>';

var menuItem = $('<li class="menu__item selector">' +
    '<div class="menu__ico">' + icon + '</div>' +
    '<div class="menu__text">Рандом</div>' +
    '</li>');

// Приклад 2: Кнопка в шапці
var buttonIcon = '<svg height="24" viewBox="0 0 24 24" width="24">' +
    '<path d="M12 2C6.48 2 2 6.48 2 12s4.48 10 10 10..." fill="currentColor"/>' +
    '</svg>';

var button = $('<div class="head__action selector">' + buttonIcon + '</div>');

// Приклад 3: В налаштуваннях
Lampa.SettingsApi.addComponent({
    component: 'my_plugin',
    icon: icon,
    name: 'Мій плагін'
});
```

---

**Примітка:** Всі іконки використовують `fill="currentColor"` для автоматичного підбору кольору під тему Lampa.
